**DNAV (Decarbonized Net Asset Value)** is a state-of-the-art audit-grade carbon-finance engine designed to meet the rigorous regulatory landscape of April 2026. It bridges the critical gap between traditional Private Equity accounting and the "Reasonable Assurance" requirements of **SFDR 2.0 (EU)** and **BRSR Core (India)**.

---

## 🎯 Mission Statement
To provide institutional investors, auditors (PwC/EY/KPMG), and ESG leads with a daily, risk-adjusted portfolio visibility that transforms "Carbon Data" into "Financial Reality."

## 🛠️ Key Modules

### 1. 📊 Portfolio Carbon Navigator (Dashboard)
A real-time visual interface comparing **Gross NAV** vs. **Carbon-Adjusted NAV (caNAV)**. Features 2030 Temperature alignment projections and interactive Plotly charting.

### 2. 🧪 AI Net-Zero Rebalancer
A stochastic optimization engine that suggests capital re-allocation to hit Net-Zero targets while minimizing portfolio variance.

### 3. 🛡️ GRC Sentry (AI Oversight Hub)
A high-integrity control center for reviewing AI-detected fraud flags. Includes a mandatory **AI Kill Switch** for manual oversight if variance triggers exceed 5%.

### 4. 🌐 End-to-End Data Lineage
Full visual traceability using **Graphviz**, mapping data from Upstream (SAP S/4HANA/Utility APIs) to Downstream (Audit Vaults/SFDR Registries).

## 📡 Deployment & Development

### 🌍 Live Application
Access the production environment here: [DNAV Live](https://prakush-dnav-carbon-navigator-git.streamlit.app/)

### 💻 Local Setup
1.  **Clone the Repo**:
    ```bash
    git clone https://github.com/prakush1/DNAV-Carbon-Navigator.git
    cd DNAV-Carbon-Navigator
    ```
2.  **Install Dependencies**:
    ```bash
    pip install -r requirements.txt
    ```
3.  **Run the Engine**:
    ```bash
    streamlit run dnav_app.py
    ```
